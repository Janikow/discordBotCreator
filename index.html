⚠️ Never share your token. It's only used locally to pre-fill your .env file.
🔑 How to get your token
  1. Go to discord.com/developers/applications
  2. Click New Application → give it a name
  3. Go to the Bot tab → click Add Bot
  4. Under Token click Reset Token → copy it
  5. Enable Privileged Gateway Intents you need below
  6. Go to OAuth2 → URL Generator to invite your bot

Deploy Instructions

1Create a new Node.js Repl on replit.com
2Upload the downloaded files or paste the code into the editor
3Add your bot token in Replit Secrets (not .env) as DISCORD_TOKEN
4Run npm install then click Run 🚀
5Keep alive 24/7 — use UptimeRobot to ping your Repl URL every 5 minutes
⚠️ The generated code includes a keep-alive Express server required for Render's free tier.
1Push your downloaded files to a GitHub repository (add .env to .gitignore)
2Create a new Web Service on render.com and connect your GitHub repo

4Add DISCORD_TOKEN under Environment → Secret Files or Environment Variables
5Click Deploy — then use UptimeRobot to ping your Render URL every 14 minutes to prevent sleep
